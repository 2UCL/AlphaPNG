# AlphaPNG(proto) for Image tool Windows App
For Windows only - x86,x64. These files cannot be run in DOS mode.

【重要】
  対象画像データをあるだけ処理するので高確率でウイルス対策ソフトとかに検知されます(もともと自作ソフトはウイルス対策ソフトの餌食になるけど)

【なにこれ】
  透明色指定(上限下限)して、フォルダアドレス指定してOKを押すと指定されたフォルダの中の画像(厳密にいえばファイル全部)を透明化処理して「worked」フォルダを作りpngとして保存してくれます。
  ファイル全部と言ってるだけあるので読み取り可能外ファイルをフォルダに入れてあるとエラーを出して終了します。

【その他】
